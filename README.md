# VirtualKeyboard

一个基于 WPF（.NET 6）的桌面虚拟键盘。

## 功能

- 无边框置顶窗口，支持拖拽移动与缩放
- 可切换按键配色主题（含异型 Enter 键）
- 拖拽条提供字体大小快速调节
- 支持部分按键长按重复（如 Backspace / 字母 / 数字 / 符号）

## 开发环境

- Windows
- .NET SDK 6.0+

## 运行

```powershell
dotnet build
dotnet run
```

## 解决方案/项目文件

- `VirtualKeyboard.sln`
- `VirtualKeyboard.csproj`
